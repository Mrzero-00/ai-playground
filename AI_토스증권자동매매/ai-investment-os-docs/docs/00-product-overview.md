# Product Overview

## 제품 정의

이 시스템은 단순한 차트 패턴 봇이 아니다.

```text
시장 전체 탐색
→ 증거 수집
→ 미래가치 가설 생성
→ 가격 반영 정도 평가
→ 통계적 진입 검증
→ 결정론적 리스크 승인
→ 페이퍼 또는 실거래
→ 거래 사후 평가
→ 검증된 전략 개선
```

## 핵심 목표

- 미국 시장 전체에서 조사할 가치가 있는 후보를 자동 발견한다.
- 공식 자료와 시장 데이터를 하나의 Evidence Bundle로 구성한다.
- 미래 현금흐름이나 수급 구조를 바꾸는 사건을 평가한다.
- 이미 가격에 과도하게 반영된 종목은 거부한다.
- 틀렸을 때 계좌가 생존하도록 손실을 제한한다.
- 수익 결과와 의사결정 품질을 분리해 평가한다.

## 비목표

- 자체 LLM 훈련
- LLM에 계좌 리스크 통제 위임
- 실시간 전략 자기 수정
- 뉴스 한 건만으로 자동매수
- 레버리지 기반 수익 극대화
- 손절 없는 물타기
- Phase 1 실제 주문
- ChatGPT 예약 알림을 주문 트리거로 사용

## 전략 영역

### Core
수개월~수년. 실적, 가이던스, 사업 품질, 밸류에이션 중심.

### Momentum
수일~수개월. 실적 상향, 업종 상대강도, 거래량과 추세 중심.

### Event / Dopamine
수분~3거래일. FDA, M&A, 역합병, 스핀오프, 대형 계약, 저유통주 이벤트 중심.

각 전략은 데이터, 손절, 보유 기간, 평가 기준을 공유하지 않는다.
