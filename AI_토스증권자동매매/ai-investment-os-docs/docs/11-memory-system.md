# Memory System

메모리는 세 종류로 분리한다.

## Episodic Memory
개별 사건과 거래.

예: 특정 기업 실적, FDA 승인, 역합병 거래.

## Semantic Memory
여러 사례에서 반복 검증된 일반 규칙.

예: 비구속적 LOI는 종가 유지율이 낮음.

## Regime Memory
시장 국면과 전략 성과.

예: Risk-off에서 프리마켓 추격 전략 성과 악화.

## 주의

- 단일 사건을 일반 규칙으로 승격하지 않는다.
- 기억에는 근거 거래와 표본 수를 연결한다.
- 오래된 규칙은 재검증한다.
- 현재 시장과 과거 국면 유사도는 참고 신호이며 진입 근거가 아니다.
