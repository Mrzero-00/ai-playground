# AI Policy

## AI가 담당하는 일

- 뉴스·공시 요약
- 사건 유형 분류
- 미래가치 변화 가설 생성
- 숨은 위험과 가정 추출
- Bull/Bear/Skeptic 분석
- 사후 정성 평가
- 전략 변경안 초안 작성

## AI가 담당하지 않는 일

- 주문 수량 최종 결정
- 계좌 위험 한도 변경
- Kill Switch 해제
- 실거래 전략 자동 승격
- 실시간 자기 코드 변경
- 증거 없는 사실 생성

## 출력 규칙

- Structured Output만 사용
- Zod 검증 실패 시 재시도 후 거부
- 근거 Evidence ID를 함께 반환
- 모르는 내용과 누락된 증거를 명시
- 상관관계를 인과관계로 표현하지 않음
- 같은 입력과 전략 버전은 재현 가능해야 함
