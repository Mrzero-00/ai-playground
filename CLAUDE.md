# AI Playground — 루트 가이드

이 저장소는 AI를 활용한 여러 프로젝트의 모음입니다. **각 폴더가 하나의 독립된 프로젝트**입니다.

## 폴더 = 프로젝트 규칙

- 새 프로젝트는 루트 아래 새 폴더로 추가한다.
- 각 프로젝트 폴더에는 다음을 둔다.
  - `README.md` — 상단에 반드시 `> 한 줄 요약` (블록인용구) 을 둔다.
  - `CLAUDE.md` — 그 프로젝트에만 적용되는 Claude 설정/컨텍스트.
- 프로젝트별 작업은 해당 폴더의 `CLAUDE.md` 를 우선 컨텍스트로 삼는다.

## 루트 README 유지 규칙

루트 `README.md` 의 "프로젝트 목록" 섹션은
`<!-- PROJECTS:START -->` ~ `<!-- PROJECTS:END -->` 사이가 **자동 생성 영역**이다.
각 프로젝트 README의 `>` 인용구를 요약으로 사용한다.

**프로젝트를 추가/삭제하거나 프로젝트 README의 요약 인용구를 바꾼 뒤에는
반드시 아래를 실행해 루트 README를 갱신한다.**

```bash
node scripts/update-readme.mjs
```

- 자동 생성 영역(마커 사이)은 직접 수정하지 않는다. 스크립트로만 갱신한다.
- 마커 바깥의 내용은 자유롭게 편집해도 된다.

## 새 프로젝트를 만들 때 체크리스트

1. 루트에 프로젝트 폴더 생성.
2. 그 폴더에 `README.md` 작성 — 상단에 `> 한 줄 요약`.
3. 그 폴더에 `CLAUDE.md` 작성 — 프로젝트 목표/스택/규칙.
4. `node scripts/update-readme.mjs` 실행해 루트 README 갱신.
