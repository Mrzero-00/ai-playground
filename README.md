## 구조 규칙

- 새 프로젝트는 루트 아래 폴더로 추가합니다.
- 각 프로젝트 폴더에는 `README.md` 와 `CLAUDE.md` 를 둡니다.
- 각 프로젝트 `README.md` 상단에는 한 줄 요약을 `> 인용구` 형식으로 작성합니다.
  이 인용구가 위 "프로젝트 목록" 요약에 자동 반영됩니다.

## 프로젝트 목록 갱신

프로젝트를 추가/변경한 뒤 아래 명령으로 루트 README의 목록을 갱신합니다.

```bash
node scripts/update-readme.mjs
```
