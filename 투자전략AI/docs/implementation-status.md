# 구현 상태

## 현재 기준

- 장기 투자와 모멘텀 점수 모델을 별도 모듈과 별도 입력 타입으로 분리
- 장기 80~90%, 모멘텀 10~20% 정책을 포트폴리오 엔진에서 검증
- 의사결정, 결과, 교훈, 모델 버전을 저장할 저장소 계약 정의
- 외부 프레임워크 없이 구동되는 HTTP API 제공

가중치와 판정 임계값은 상세 명세가 비어 있어 초기 가설로 설정했습니다. 제품 정책이 정해지면 모델 버전과 함께 변경해야 합니다.
